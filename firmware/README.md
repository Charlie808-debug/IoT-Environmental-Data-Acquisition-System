# Firmware

The firmware for this project was developed using the Arduino IDE on the ESP32 platform.

## Functionality

- Initializes BMP280, APDS9930, ADS1115, and UV sensor
- Collects environmental parameters
- Hosts an embedded Wi-Fi web server
- Displays live sensor values through a browser-based dashboard
- Updates readings periodically in real time

## Note

The original Arduino source code is currently unavailable. This repository preserves the project documentation, hardware design, circuit connections, dashboard screenshots, and implementation report for reference.
